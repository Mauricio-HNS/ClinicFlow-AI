# Demo images (optional local override)

Place your local photos here using these names:

- `assets/demo/casa_jardim.jpg`
- `assets/demo/moda_beleza.jpg`
- `assets/demo/eletronicos.jpg`
- `assets/demo/animais.jpg`
- `assets/demo/veiculos.jpg`
- `assets/demo/imoveis.jpg`
- `assets/demo/outros.jpg`
- `assets/demo/servicos.jpg`
- `assets/demo/esportes.jpg`
- `assets/demo/infantil.jpg`

If a local file is missing, the app automatically falls back to an online demo image.
